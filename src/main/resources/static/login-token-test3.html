앱 스타일 토큰 로그인 테스트 (Google / Apple)

예) http://localhost:8080
아래 Google 버튼으로 브라우저에서 즉시 ID 토큰을 발급받아 자동 입력할 수도 있습니다.
로그인/리프레시는 선택, 로그아웃(단말 1대만) 시 반드시 포함. 테스트: 직전 로그인에서 받은 memberPk + 100

Token Control (수동 입력 / 복사)

Manual 값이 비어있으면 자동으로 저장된(access) 토큰을 사용합니다.
Manual 값이 비어있으면 자동으로 저장된(refresh) 토큰을 사용합니다.

Google (선택) — 브라우저에서 바로 ID 토큰 받기

버튼 클릭 → Google 팝업 → credential(ID 토큰) 수신 → 위 텍스트 영역에 자동 채움
